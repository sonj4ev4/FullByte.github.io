    Switch Taiko

    I built a hardware case for the original hori taiko drum controller for Nintendo Switch which is a great addition to play the game Taiko no Tatsujin: Drum 'n' Fun!.

    Components

    • Hori taiko drum controller
    • Amazon basic speakers
    • Anker Ultra Slim 4 Port USB 3.0 Hub
    • Switch Docking station
